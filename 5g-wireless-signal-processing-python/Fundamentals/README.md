# Fundamentals — Basic 5G Calculations

Starting point for understanding key 5G NR physical layer parameters.

## What this covers
- Wavelength calculation at 28 GHz mmWave frequency
- Complex baseband signal analysis (magnitude, power, phase)
- RF unit conversions (dB, dBm, Watts)

## Why these matter
At 28 GHz the wavelength is ~10.7mm which directly affects antenna element 
spacing in massive MIMO arrays. Complex signals and dB conversions are used 
in every RF measurement and link budget calculation.
